================================

DEMO TESTEN
-----------
1. Diesen Ordner mit einem lokalen Webserver öffnen:
   python -m http.server 8080
2. Im Browser öffnen: http://localhost:8080
3. Benutzername: inhaber
4. Passwort: afterhours

ECHTES SUPABASE-BACKEND
-----------------------
1. README.md vollständig lesen.
2. Öffentliche Registrierung in Supabase deaktivieren.
3. supabase/schema.sql im Supabase SQL Editor ausführen.
4. Erstes Inhaberkonto gemäß README anlegen.
5. Die Edge Functions create-employee und update-employee bereitstellen.
6. Projekt-URL und Publishable Key in assets/js/config.js eintragen.
7. demoMode auf false setzen.
8. Repository über GitHub Pages veröffentlichen.

WICHTIG
-------
Niemals einen Secret Key oder service_role-Key in assets/js/config.js eintragen.
Dieses Projekt ist nicht als fiskalisch zertifiziertes Kassensystem ausgewiesen.
